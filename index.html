Fractal Benchmark

Fractal Benchmark is a web based Mandelbulb renderer implementation with ray marching, It used webGL to perform ray marching on a Mandelbulb fractal algorithm.

Fractal Benchmark renders a beautiful fractal of the famous mandelbuld which is a 3D implimentation of the mandelbrot set. As it used the client to render its highly graphics intensive render it can be used to perform benchmarking for the system it is ran on.

Along side all these I have also implimented a UI to make it more interactive and capable of creating more intresting shapes.
